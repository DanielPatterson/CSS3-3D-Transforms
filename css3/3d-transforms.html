CSS3 3D Transforms

Overview

Using 3D transformations to transform elements on the X, Y and Z axis.

Demo


  • rotateX(180deg)

  • rotateY(120deg)

  • rotateZ(90deg)

Code

  • html
  • CSS

<img src="../../img/cat-specs.png" class="img-thumbnail rotateX" />
<img src="../../img/cat-specs.png" class="rotateY" />

.rotateX {
-webkit-transform: rotateX(180deg); /* Safari */
transform: rotateX(180deg);
}	 

.rotateY {
-webkit-transform: rotateY(120deg); /* Safari */
transform: rotateY(120deg);
} 

.rotateZ {
-webkit-transform: rotateZ(90deg); /* Safari */
transform: rotateZ(90deg);	
}
